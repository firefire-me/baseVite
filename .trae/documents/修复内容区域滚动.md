# 修复内容区域滚动问题

## 修改文件
- `src/components/MainLayout.vue`

## 具体修改
将 `.content` 的样式从 `min-height` 改为固定 `height`，并添加滚动属性：

```less
.content {
  margin-left: 200px;
  transition: margin-left 0.2s;
  background: #f0f2f5;
  height: calc(100vh - 64px);      // 改为固定高度
  overflow-y: auto;                  // 添加垂直滚动
  overflow-x: hidden;                // 隐藏水平滚动

  .content-wrapper {
    padding: 24px;
    min-height: 100%;               // 确保内容至少填满
  }
}
```

## 效果
- 侧边栏固定，不滚动
- 顶部导航栏固定，不滚动
- 只有内容区域滚动
