# 添加用户个人中心和头像上传功能

## 实现步骤

### 1. 创建个人中心组件
- 创建 `src/components/Profile.vue` 组件
- 实现头像上传和预览功能
- 设计个人中心页面布局

### 2. 配置路由
- 在 `src/router/index.ts` 中添加个人中心的子路由

### 3. 更新导航栏
- 在 `src/components/Home.vue` 中添加个人中心菜单项
- 更新路由监听逻辑，确保菜单项正确选中

### 4. 实现头像上传功能
- 使用 Ant Design Vue 的 `Upload` 组件实现头像上传
- 添加头像预览功能
- 使用 localStorage 存储头像 URL

## 技术要点

- 使用 Vue 3 Composition API
- 使用 Ant Design Vue 的 `Upload` 组件处理文件上传
- 使用 localStorage 存储用户头像信息
- 实现响应式布局设计

## 预期效果

- 用户可以通过导航栏进入个人中心
- 用户可以上传、预览和修改头像
- 头像信息会保存在本地存储中
- 页面布局美观，交互友好
