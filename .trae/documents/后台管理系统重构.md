# 后台管理系统重构计划

## 1. 布局重构
- 创建 `MainLayout.vue` 组件（左侧侧边栏 + 顶部导航栏）
- 侧边栏支持折叠/展开，包含菜单图标和文字
- 顶部导航栏包含用户头像、下拉菜单、通知图标

## 2. 首页 Dashboard 重构
- 创建 4 个数据统计卡片（任务总数、已完成、待完成、完成率）
- 添加任务趋势折线图（使用 ECharts 或 Ant Design Vue 图表）
- 添加任务状态分布饼图
- 添加快捷操作按钮区
- 添加最近活动列表

## 3. 登录/注册页面优化
- 采用分屏设计（左侧品牌宣传区 + 右侧表单区）
- 添加渐变背景和动画效果
- 优化表单样式和交互

## 4. 新增功能组件
- 创建 `BreadcrumbNav.vue` 面包屑导航组件
- 创建 `StatCard.vue` 统计卡片组件
- 创建 `ChartContainer.vue` 图表容器组件

## 5. 路由配置更新
- 更新路由结构，适配新的布局
- 添加面包屑配置

## 6. 样式优化
- 统一配色方案
- 添加过渡动画
- 优化响应式布局
