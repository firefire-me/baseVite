# 为任务中心添加分页和搜索功能

## 需求分析
用户要求为任务中心添加分页和搜索功能，API 返回的数据格式包含分页信息，支持的查询参数有：
- page: 当前页码
- limit: 每页条数
- status: 任务状态（true/false）
- search: 搜索关键词

## 实现计划

### 1. 修改 API 函数
- 更新 `src/api/task/index.ts` 文件中的 `getTasks` 函数，支持传入查询参数
- 定义 API 返回数据的类型接口

### 2. 添加状态管理
- 在 `TaskCenter.vue` 中添加分页相关的状态变量：currentPage, pageSize, total, totalPages
- 添加搜索相关的状态变量：searchText, statusFilter

### 3. 更新 UI 组件
- 添加搜索表单，包含搜索输入框和状态筛选
- 修改表格组件，启用分页功能
- 更新表格数据绑定

### 4. 修改加载逻辑
- 更新 `loadTasks` 函数，支持传入分页和搜索参数
- 在分页或搜索条件变化时重新加载数据

## 详细步骤

1. **修改 `src/api/task/index.ts`**：
   - 为 `getTasks` 函数添加查询参数类型定义
   - 定义 API 响应数据的类型接口
   - 更新 `getTasks` 函数实现，支持传入查询参数

2. **更新 `TaskCenter.vue` 中的状态管理**：
   - 添加分页状态变量
   - 添加搜索状态变量

3. **更新 `TaskCenter.vue` 中的 UI**：
   - 在表格上方添加搜索表单
   - 修改表格的 pagination 属性，启用分页

4. **修改 `loadTasks` 函数**：
   - 接收分页和搜索参数
   - 调用更新后的 `getTasks` 函数
   - 处理 API 返回的分页数据

5. **添加事件处理**：
   - 搜索按钮点击事件
   - 分页变化事件
   - 状态筛选变化事件

## 预期结果
- 任务列表支持分页显示
- 可以通过搜索框进行模糊搜索
- 可以通过状态筛选任务
- 分页组件显示正确的总页数和当前页码
- 每页显示数量可配置
